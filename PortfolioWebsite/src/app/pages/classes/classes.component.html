Web Application Development II CMP_SC 4830 This course studies the science and engineering of the World Wide Web. Covers the languages, protocols, services and tools that enable the web. Emphasis on the on basics and technologies.
Object Oriented Programming CMP_SC 3330 This course focuses on object-oriented programming concepts: abstraction, polymorphism, encapsulation, inheritance, interfaces, abstract classes, files, streams, and object serialization. Software Engineering INFOTC 4320 Software Engineering covers the principles, processes, and professional practices used to design, develop, test, deploy, and manage software systems in a team-based, collaborative environment. Emphasis is placed on techniques for efficient implementation and good software development methodologies. Written mainly in C.
